IUNOIA CORE — FULL MODEL RUN GUIDE

---

1. OPEN TERMINAL

Use Command Prompt or PowerShell.

---

2. NAVIGATE TO PROJECT ROOT

You should see:

* run.py
* generate_datasets.py
* model/
* config/
* data/
* demo/

---

3. INSTALL REQUIREMENTS

Run:

py -m pip install -r requirements.txt

---

4. VERIFY PROJECT FILES

Ensure these exist:

config\constants.json
demo\sample_input.json
model\predictor.py
model\features.py
model\risk_scores.py
generate_datasets.py
run.py

---

5. (OPTIONAL) GENERATE SYNTHETIC TRAINING DATA

Only needed if you want to train the ML model and don't have real data.

Run with defaults (generates seed, scenario, and parameter variants):

py generate_datasets.py

Output is written to three subfolders:

data\synthetic\seeds\        — seed variants (5 by default)
data\synthetic\scenarios\    — mission risk profiles (low/moderate/high/worst-case)
data\synthetic\params\       — weight combos from constants.json (8 variants)

Options:

py generate_datasets.py --seeds 3          # change number of seed variants
py generate_datasets.py --no-seeds         # skip seed variants
py generate_datasets.py --no-scenarios     # skip scenario variants
py generate_datasets.py --no-params        # skip parameter sweep variants
py generate_datasets.py --out my_data\     # write to a custom folder

---

6. (OPTIONAL) TRAIN MACHINE LEARNING MODEL

Only needed if you have a dataset (real or generated).

Step 6.1 — Place dataset:

data\raw\your_data.csv

Or use a file generated in Step 5, e.g.:

data\synthetic\seeds\seed_001.csv

Dataset must include target columns:
cycle, bone, cortisol

Step 6.2 — Train model:

py -m model.train

This creates:

model\saved\model.pkl

---

7. RUN FULL MODEL (USING run.py)

From project root:

py run.py

---

8. WHAT THE SYSTEM DOES

Flow:

sample_input.json
↓
build_features()
↓
predict_features()
↓              ↓
ML model     simulator fallback
↓
risk_scores.py
↓
FINAL OUTPUT

---

9. EXPECTED OUTPUT

---

10. COMMON ERRORS + FIXES

ERROR: ModuleNotFoundError
Fix:
py -m pip install -r requirements.txt

ERROR: FileNotFoundError: demo/sample_input.json
Fix:
Ensure file exists:
iunoia-core\demo\sample_input.json

ERROR: constants.json not found
Fix:
Ensure file exists:
iunoia-core\config\constants.json

ERROR: Model load failed
Fix:
Train model (Step 6) OR ignore — simulator will run as fallback

ERROR: ML model failed, falling back
Fix:
Feature mismatch between saved model and current dataset.
Regenerate data (Step 5) and retrain (Step 6).

ERROR: cortisol_load not defined
Fix:
Ensure predictor.py includes:

cortisol_score_0_to_1 = _clamp(base_cort * time_gain * sensitivity, 0.0, 1.0)
cortisol_load = round(100.0 * cortisol_score_0_to_1, 1)

ERROR: FileNotFoundError: data/synthetic/...
Fix:
Run generate_datasets.py first (Step 5)
